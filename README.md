# BookApp API

## Technologies Used
* ExpressJS
* PostgreSQL
* Sequelize

## How to use
* `git clone` https://github.com/meetKazuki/BookApp-API.git
* `cd` to _BookApp-API_
* `npm install`
* `npm run start:dev` to start server and watch for changes
* _`npm test` to run automated tests_
